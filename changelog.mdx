## How to Upgrade

Upgrading keeps you on the latest version. Follow these steps to update your Anastasiia Konevytska instance.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your project root.

````bash
npx anastasiia-version
````

  </Step>
  <Step title="Update Package" icon="download">
    Install the latest version using your package manager.

    <CodeGroup tabs="npm,yarn">
````bash
npm install anastasiia-konevytska@latest
````
````bash
yarn add anastasiia-konevytska@latest
````
    </CodeGroup>
  </Step>
  <Step title="Rebuild Docs" icon="refresh-cw">
    Rebuild your documentation site.

````bash
npm run build
````

  </Step>
  <Step title="Verify Changes" icon="check-circle">
    Test key pages like this changelog for new components.
  </Step>
</Steps>
