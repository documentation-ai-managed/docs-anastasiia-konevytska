## Document Creation and Editing

You create new documents instantly from the dashboard. Choose from templates or start blank. The rich text editor supports Markdown, embeds, and syntax highlighting for code blocks.

<Steps>
  <Step title="Create Document" icon="plus">
    Click the `New Document` button. Enter a title like `API Reference` and select a folder.
  </Step>
  <Step title="Edit Content" icon="edit">
    Use the editor toolbar for headings, lists, and images. Insert code with `Ctrl + K`.
  </Step>
  <Step title="Preview and Publish" icon="eye">
    Toggle preview mode. Save changes or publish to make live.
  </Step>
</Steps>

Advanced editing includes version history. You revert changes or compare diffs easily.

## Project Organization and Folders

Organize large projects with nested folders. Create hierarchies like `Docs > API > Endpoints` for clarity.

| Feature | Description | Benefit |
|---------|-------------|---------|
| Nested Folders | Up to 10 levels deep | Scalable structure |
| Search & Filters | Full-text search | Quick navigation |
| Bulk Move | Drag-and-drop | Efficient reorganization |

Drag documents between folders. Tags add cross-folder discoverability.

## Collaboration and Sharing Options

Invite team members for collaborative editing. Control permissions per document or project.

<Tabs>
  <Tab title="Public Sharing" icon="globe">
    Generate shareable links. Set to view-only or editable.

    ```
    https://docs.anastasiia-konevytska.com/project/doc-id
    ```
  </Tab>
  <Tab title="Team Invites" icon="users">
    Add users by email. Roles: Viewer, Editor, Admin.
  </Tab>
  <Tab title="Embed" icon="code">
    Embed docs in sites.

````html
<iframe src="https://docs.anastasiia-konevytska.com/embed/doc-id" width="100%" height="600"></iframe>
````

  </Tab>
</Tabs>

Real-time cursors show who edits what.

## Customization with Brand Colors

Tailor your docs to match your brand. Set primary colors in project settings.

<CodeGroup tabs="JSON,CSS">
```json
{
  "theme": {
    "primary": "#3B82F6",
    "secondary": "#1E40AF"
  }
}
```
```css
:root {
  --primary-color: #3B82F6;
  --accent: #3B82F6;
}
```
</CodeGroup>

<Callout kind="info">
  Apply `#3B82F6` as your brand color for consistency across all docs.
</Callout>
