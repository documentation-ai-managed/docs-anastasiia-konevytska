## Overview

Anastasiia Konevytska provides a centralized space for you to organize, manage, and collaborate on project documentation. You create custom documentation hubs tailored to your team's needs, with support for rich MDX content, version control, and seamless search. Whether you document APIs, user guides, or internal wikis, this platform streamlines your workflow and ensures knowledge stays accessible.

## Quick Start

Get up and running in minutes with these steps.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://app.anastasiia.com` and create a new documentation space. Choose your brand color, such as `#3B82F6`.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Click "New Page" and select the `introduction.mdx` template. Start editing with MDX syntax.
  </Step>
  <Step title="Publish and Share" icon="globe">
    Save and publish. Share the live URL with your team via `https://docs.anastasiia.com/your-space`.
  </Step>
</Steps>

For example, embed a code snippet in your docs:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch documentation data
  const response = await fetch('https://api.anastasiia.com/docs');
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  # Fetch documentation data
  import requests
  response = requests.get('https://api.anastasiia.com/docs')
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
